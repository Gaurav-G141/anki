# Physics GRE — Anki Deck Manifest (one deck per topic)

The single best **AnkiWeb** shared deck for each Physics GRE subject. AnkiWeb
blocks anonymous/bulk downloads (`429 "Please log in to download more decks"`),
so these are **not auto-downloaded** — download each yourself.

**Every AnkiWeb ID below was verified** against the anki-decks.com mirror (which
links to the real AnkiWeb page), so the links resolve and card counts are real.

## How to download each

- **In a browser (logged into AnkiWeb):** open the info link → click **Download**
  → you get a `.apkg`.
- **In Anki desktop:** the info page's Download gives a `.apkg`; then
  **File → Import**. (Save them into this `anki-decks/` folder if you want them
  co-located.)

## Honest caveats — read before relying on these

1. **¹ Four topics resolve to the same "Physics GRE" deck (ID 1148323968).**
   AnkiWeb has **no dedicated, GRE-level deck** for Classical Mechanics,
   Thermodynamics/Stat-Mech, Atomic Physics, or Laboratory Methods. The
   PGRE-wide deck (top-rated physics deck on the mirror, 7 ratings) is the best
   available pick for each — but it's one 108-card deck covering all of them, not
   four separate large decks. (This matches the forum research finding that
   PGRE-specific material is scarce.)
2. **² The "CBSE Class XII" decks (E&M, Optics, Modern Physics) are Indian
   high-school level**, not calibrated to PGRE depth. They're the **largest
   topic-specific** decks available (E&M has 347 cards), good for fundamentals
   and vocabulary, but they will _not_ fully cover GRE-level problems.
3. **³ Special Relativity has no standalone deck.** "Modern Physics CBSE Class
   XII" is the closest verifiable pick (SR is one section alongside photoelectric
   effect, atoms, nuclei). A more advanced but ID-unverified option is "Cambridge
   Part II Physics: Relativity" — search AnkiWeb by title if you want depth.
4. **⁴ "Specialized Topics" is a catch-all** (nuclear, particle, condensed
   matter, astro). The Particle Physics deck (58 cards, best-rated of the
   specialized subtopics) covers only part of it; the Physics GRE deck (backup)
   spans the bucket more broadly but shallowly.

**Bottom line:** genuinely strong, GRE-calibrated topic decks only really exist
for **Electromagnetism** and **Quantum Mechanics**. The rest are best-available
compromises — a general PGRE deck or a high-school-level topic deck — because
AnkiWeb simply doesn't host dedicated GRE-depth decks for those subjects.
