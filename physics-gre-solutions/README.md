# Physics GRE — Problems & Solutions

A comprehensive, scraped compilation of every problem and worked solution from
the four officially released **GRE Physics (PGRE)** practice exams, sourced from
[grephysics.net](https://grephysics.net/ans/all-solutions_list.php).

**400 problems total — 4 exams × 100 problems each.**

## Format

Each problem entry contains:

```
## Problem N
**Topic:** <subject area, e.g. Mechanics → Newtonian>
**Answer:** <correct choice letter>

<problem statement>

**Solution:** <full worked solution with LaTeX-style $...$ math>
```

## Source & accuracy notes

- Source: `https://grephysics.net/ans/{exam}/{1..100}` — a community-maintained
  solution wiki for the released PGRE exams.
- On grephysics.net the **verbatim question text is usually embedded as an
  image** (and some pages are marked "still being typed"). Where the literal
  statement wasn't available as text, the problem statement here was
  faithfully **reconstructed** from the given values, answer choices, and the
  worked solution on the page.
- A small number of pages did not expose an explicit answer **letter**; for
  those the correct value/result is recorded instead, and the omission is noted
  inline. Per-exam caveats:
  - **GR9677 #90** — an ETS unscored item; no statement or solution exists.
  - **GR9277 #53, #55, #82** — source page has a noted typo / internal
    inconsistency; keyed answer preserved with a flag.
  - **GR0177 #5, #12, #16, #23, #27, #45, #98** and
    **GR8677 #25, #52, #55, #95** — value/result captured where the letter
    wasn't shown.
- Figure/graph-dependent problems are summarized via their elimination
  reasoning rather than reproduced images.

These solutions are community-contributed and may occasionally contain errors;
treat them as study aids, not authoritative answer keys.
